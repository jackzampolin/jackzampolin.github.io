The hows and whys


14 Feb 2015

However if you want to separate your CSS off into another document so that you can easily apply your styles multiple times over a number of different pages you are faced with a couple of choices about how you can implement that: classes or IDs.

ID names are signified on the stylesheet with a '#'. They look like this:

IDs are unique identifiers and should only be used once per page. That makes them especially suited to things that only show up once on each page; like a header or footer. That is the example I use above.
